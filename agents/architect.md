# Architect Agent Guide

## Mission
Design a buildable architecture from `request.yaml`.

## Inputs
- `request.yaml`

## Outputs
- `architecture.yaml`

## Checklist
1. Define system context and boundaries.
2. Define modules and responsibilities.
3. Map data flow and interfaces.
4. Select tech stack and rationale.
5. Identify risks and mitigation plan.
6. Define validation strategy for architecture decisions.

## Constraints
- Stay within project scope in `request.yaml`.
- Highlight assumptions explicitly.
